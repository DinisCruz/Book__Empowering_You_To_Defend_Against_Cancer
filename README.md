# Taking Control: A Lifestyle Medicine Approach to Cancer Prevention and Treatment

**By Deb Brewer**

## About This Book

This book documents Deb Brewer's personal journey as a Stage 4 cancer survivor and presents her ReachVitality 6-Pillar Lifestyle Medicine Program for preventing and fighting cancer. Through a combination of personal experience, Harvard Medical School training in Lifestyle Medicine, and research from leading institutions, this book empowers readers to take control of their health through evidence-based lifestyle interventions.

## Publishing on LeanPub

This book is written in Markua format and published using LeanPub's GitHub integration.

### Structure

- `manuscript/` - Contains all book content in Markua format
- `manuscript/Book.txt` - Defines the chapter order
- `manuscript/frontmatter.md` - Title page, copyright, and table of contents
- `manuscript/chapter*.md` - Individual chapters
- `manuscript/backmatter.md` - Sources, about the author, and disclaimer

### How to Publish

1. Connect this GitHub repository to your LeanPub book
2. Configure LeanPub to use the GitHub writing mode
3. Set the manuscript folder to `manuscript`
4. Push changes to the `main` branch to trigger a new book generation
5. LeanPub will automatically generate PDF, EPUB, and MOBI versions

### Local Preview

To preview the book locally, you can use any Markdown viewer or the LeanPub command-line tools.
